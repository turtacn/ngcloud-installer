ngcloud-installer
========

Repo for building the nghci/ngcloud ISO image.

0-1 to build cloud, like `harvester`

## Building

Built ISO image is located in the `dist/artifacts` directory.

```shell script
./build
```

In `build` script, there are ways to change proxy settings.
